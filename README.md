# TMEB Consulting - Website

Site institucional minimalista e moderno para a TMEB Consulting, empresa de consultoria de tecnologia.

## Características

- Design minimalista e inovador
- Animações suaves e interativas
- Totalmente responsivo
- Foco em visual impactante
- Seção de contato destacada

## Estrutura

- `index.html` - Estrutura principal do site
- `styles.css` - Estilos e animações
- `script.js` - Interatividade e efeitos

## Como usar

1. Abra o arquivo `index.html` em um navegador moderno
2. Ou sirva via servidor local:
   ```bash
   python -m http.server 8000
   ```
   E acesse `http://localhost:8000`
